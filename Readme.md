# No-Code Projects

A collection of projects built entirely with AI using natural language prompts—**without writing any code manually**.

The purpose of this repository is to explore how capable modern AI models are at developing real-world applications through prompt engineering alone.

## About

Every project in this repository is generated by AI. The goal is to avoid manual coding as much as possible and rely on AI to build complete applications from prompts.

Some projects may require multiple prompts and iterations, but no code is written directly by the developer unless explicitly stated.

## Repository Structure

Each project is placed in its own folder with a dedicated `README.md` describing:

* The project's purpose
* The AI model used
* Features
* Current limitations
* Future improvements
