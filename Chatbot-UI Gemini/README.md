# Chatbot UI

A modern chatbot interface created entirely using **Google Gemini** through prompt-based development. This project was built **without writing any code manually**—every line of code was generated by Gemini based on natural language instructions.

> **Note:** The generated code is currently unorganized and exists in a single file. Refactoring and restructuring are planned for future updates.

---

## Features

* Modern chatbot interface
* Light and Dark mode
* Glassmorphism-inspired design
* Supports multiple chatbot providers
* Add one or multiple API keys
* Automatically fetches available models after an API key is added
* Easily switch between available models
* Supports multiple chatbots using different API keys

---

## How It Works

This application **requires an API key** to function.

1. Add your API key for a supported AI provider.
2. The application automatically retrieves all models available for that API key.
3. Select the model you want to use.
4. Start chatting.

Without a valid API key, the chatbot cannot generate responses.

---

## Current Limitations

Since the project was generated entirely by Gemini:

* No code was written manually.
* Most of the generated code is placed in a single file.
* The codebase is not yet organized into reusable components or folders.
* Maintaining or extending the project is currently more difficult than a properly structured application.

This repository serves as an experiment in building an application using AI-generated code with minimal manual intervention.
